# Personal Todo List and Habit Tracker

Hi! This is my personal project. I am building a Command Line (CLI) application in Python to help manage daily tasks and build long-term habits using streaks.

I designed the logic and features myself because I wanted a tool that fits my exact needs.

## Features I am working on:

1. **Add Task:** Save new tasks and habits to a JSON file.
2. **Mark Task Done:** Change task status with visual colors (Green for done, Red for not done).
3. **View Calendar:** A simple text calendar to see tasks on different days.
4. **Remove Task:** Delete tasks from the list.
5. **View Streaks:** Track how many days in a row a habit is completed.

## Why JSON?

I chose JSON for data storage because it keeps the data simple, human-readable, and easy to load every time the program starts.

## How to Run

Right now, the main menu is ready. You can run it using:

```bash
python tracker.py
```
